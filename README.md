# WebTerminal demo with Golang and Xterm.js

## 简介 / Brief

这是一个 `WebTerminal` 的项目演示，使用 `Golang` 和 `Xterm.js` 实现。This is a demo for `WebTerminal`, based on `Golang` and `Xterm. js` .



## 用法 / Usage

```shell
chmod +x webterminal_xxxx
./webterminal_xxxx
```
